Gerador de Emojis

Gere exatamente n emojis e copie com 1 clique.

Como publicar

Crie sua mensagem só com emojis

Informe a quantidade de emojis e clique em Gerar.

Dica: pressione Enter para gerar rapidamente. Repetição de emojis é permitida.

Feito com ❤️ em HTML + Tailwind. Funciona offline.

Como publicar no GitHub Pages
  1. Crie um repositório no GitHub (ex.: gerador-emojis).
  2. Adicione este arquivo como index.html, faça commit e push.
  3. No GitHub: Settings → Pages → Deploy from a branch → selecione a branch (ex.: main) e a pasta /root.
  4. Aguarde a URL do Pages (ex.: https://seuusuario.github.io/gerador-emojis/).
